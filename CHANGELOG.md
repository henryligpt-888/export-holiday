## [1.0.0] - 2024-11-23

### 新增
- ✨ 初始版本发布
- 📅 支持 7 个主要国际节假日数据
- 🔍 按年份、地区、类型筛选功能
- 💼 每个节假日包含外贸业务影响分析
- 📝 提供可执行的外贸操作建议
- 🎉 中英文祝福语模板
- 🚫 标识是否适合发送商务开发信
- 📱 响应式设计，支持移动端访问

### 覆盖节假日
- 圣诞节 (Christmas Day)
- 春节 (Chinese New Year)
- 感恩节 (Thanksgiving Day)
- 斋月 (Ramadan)
- 元旦 (New Year's Day)
- 独立日 (Independence Day)
- 国际劳动节 (Labor Day)

### 技术特性
- Next.js 14 框架
- TypeScript 5 类型安全
- Tailwind CSS 3.4 样式系统
- Lucide React 图标库
- AI Studio 部署支持

## 版本说明

### 主版本号 (Major)
当进行不兼容的 API 或数据结构变更时递增。

### 次版本号 (Minor)
当以向后兼容的方式添加新功能时递增。

### 修订号 (Patch)
当进行向后兼容的问题修正时递增。

## 变更类型

- **新增 (Added)** - 新功能
- **变更 (Changed)** - 现有功能的变更
- **废弃 (Deprecated)** - 即将移除的功能
- **移除 (Removed)** - 已移除的功能
- **修复 (Fixed)** - Bug 修复
- **安全 (Security)** - 安全相关的修复
